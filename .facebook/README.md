# Facebook Metadata Files

This directory contains metadata and integration information for Facebook's use of OCamlFormat.

## Files

### project_metadata.md
General information about OCamlFormat as it relates to Facebook open source projects, including:
- Project description and classification
- Integration with Facebook's Infer project
- Dependencies and compatibility notes
- Support resources

### integration_guide.md
Comprehensive guide for integrating OCamlFormat into Facebook projects:
- Installation instructions
- Configuration recommendations
- CI/CD integration examples
- Editor setup for Facebook's development environment
- Best practices and troubleshooting

### project_config.json
Machine-readable project configuration and metadata in JSON format:
- Project details and licensing
- Facebook integration information
- Versioning and dependency information
- Metrics and compliance data
- CI/CD configuration details

## Purpose

These files serve multiple purposes:
1. **Documentation**: Provide Facebook-specific integration guidance
2. **Metadata**: Store structured information about the project's use at Facebook
3. **Compliance**: Track licensing, security scanning, and export compliance
4. **Automation**: Enable automated tooling to understand project configuration

## Usage

### For Facebook Employees
Refer to these files when:
- Integrating OCamlFormat into a new project
- Updating OCamlFormat versions
- Troubleshooting formatting issues
- Setting up CI/CD pipelines

### For Open Source Contributors
These files demonstrate:
- How large organizations integrate OCamlFormat
- Best practices for enterprise adoption
- Example configurations and workflows
- Integration patterns with other tools

## Maintenance

These files are maintained by the Facebook Open Source team in coordination with the OCamlFormat community. Updates should reflect:
- Current best practices
- Version compatibility information
- New integration patterns
- Lessons learned from production usage

For questions or updates, contact the Facebook Open Source team or file an issue on the OCamlFormat GitHub repository.
