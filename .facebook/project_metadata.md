# Facebook Open Source Project Metadata

## Project Information
- **Project Name**: OCamlFormat
- **Description**: A tool to format OCaml code automatically
- **Primary Language**: OCaml
- **License**: MIT
- **Category**: Developer Tools

## Integration with Facebook Projects
This project is used by Facebook's Infer project for OCaml code formatting.
See: https://github.com/facebook/infer

## Open Source Details
- **Public Repository**: https://github.com/ocaml-ppx/ocamlformat
- **Upstream Maintainer**: OCaml PPX team
- **Facebook Contact**: Open Source Team

## Dependencies
- OCaml compiler (>= 4.08)
- Dune build system
- OCaml standard library parsing tools

## Support and Resources
- **Issues**: Report issues on the GitHub issue tracker
- **Discussions**: Use the OCaml Discourse forum with #ocamlformat tag
- **Contributing**: See CONTRIBUTING.md for guidelines

## Related Projects
- **Infer**: Static analyzer for Java, C, C++, and Objective-C (uses OCamlFormat)
- **Reason**: Facebook's syntax extension for OCaml (inspiration for OCamlFormat)

## Compatibility Notes
- Works with Facebook's internal OCaml toolchain
- Compatible with standard OCaml distributions
- Tested against Facebook Infer codebase

- Regular updates coordinated with community releases
